# 隣接ペア（Adjacency Pair）

会話で、ある発話に対して特定の種類の応答が続きやすい組み合わせのことです。会話分析では、やりとりの基本単位としてよく使います。

## 例
- あいさつ → あいさつ（「おはよう」→「おはよう」）
- 質問 → 応答（「宿題出した?」→「出した」）
- 依頼 → 受諾/拒否（「手伝って」→「いいよ / 今は難しい」）

## 授業での見方
- 1つの発話だけでなく、その次に何が来るかまで見て読む。
- 期待される応答が来ない場合（沈黙・話題変更）も、分析の手がかりになる。
